# Robotika ETS - iMCLab

## Deskripsi
Proyek ini merupakan hasil praktikum Robotika menggunakan Kit iMCLab yang dikendalikan melalui Arduino, Python, dan IoT.

## Pengujian
- Arduino IDE (Serial Monitor)
- Python (PySerial)
- IoT (Smartphone)

## Isi Repository
- Arduino → kode Arduino
- Kode Python Robotik → kode Python
- Laporan → file laporan praktikum
- Video → hasil pengujian
